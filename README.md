## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/0xAdiyat/Dialogix.git
   ```
2. Install dependencies:
   ```bash
   cd dialogix
   flutter pub get
   ```
3. Add your gemini pro api key in main.dart file
   ```
   Gemini.init(apiKey: ApiKeys.gemini_api_key);
   ```
5. Run the app:
   ```bash
   flutter run
   ```
